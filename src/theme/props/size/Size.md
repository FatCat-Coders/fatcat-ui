List of size props, and example.

## 	Definition

CSS properties that we use for controlling the width and height of an element.

Property that adds to the component are:

- `w`
- `maxW`
- `minW`
- `h`
- `maxH`
- `minH`

## Usage 

```jsx
<Wrapper
	backgroundColor="primary600"
	w="100px"
	h="100px"
/>
```
